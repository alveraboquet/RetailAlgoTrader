# Types of Fundamentals in Different Markets

Below are examples of some of the primary fundamental analysis metrics for different markets. This list is not exhaustive.
If you wish to learn more about any of these topics [Investopedia](https://www.investopedia.com) is a great resource.

## Equities

- Earnings per share (EPS)
- Price-to-earnings ration (P/E)
- Projected earnings growth (PEG)
- Dividend payout ratio
- Divident yield
- Return on equity

## Forex

- Country of origin economic fundamentals
- Non-farm payroll
- Interest rates
- Inflation

## Crypto

- Hash rate
- Number of active addresses
- Transaction fees
- Market capitlization
- Liquidity and trading volume
- Circulating supply
